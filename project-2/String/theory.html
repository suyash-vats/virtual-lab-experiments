String Functions

  • charAt()

    • The charAt() method returns the character at a specified index (position) in a string.
    Example:-
    let text = "HELLO WORLD";
    let letter = text.charAt(6);
    console.log(letter); // Output: W

  • charCodeAt()

    • The charCodeAt() method returns the Unicode of the character at a specified index (position) in a string.
    Example:-
    let text = "HELLO WORLD";
    let letter = text.charCodeAt(6);
    console.log(letter); // Output: 87

  • concat()

    • The concat() method is used to join two or more strings.
    Example:-
    let text1 = "Hello";
    let text2 = "World";
    let text3 = text1.concat(text2);

  • includes()

    • The includes() method is used to check if a string contains a specified value.
    Example:-
    let text = "Hello world, welcome to the universe.";
    let result = text.includes("world");
    console.log(result); // Output: true

  • endsWith()

    • The endsWith() method is used to check if a string ends with a specified value.
    Example:-
    let text = "Hello world, welcome to the universe.";
    let result = text.endsWith("universe.");
    console.log(result); // Output: true

  • indexOf()

    • The indexOf() method returns the position of the first occurrence of a specified value in a string.
    Example:-
    let text = "Hello world, welcome to the universe.";
    let result = text.indexOf("world");
    console.log(result); // Output: 6

  • lastIndexOf()

    • The lastIndexOf() method returns the position of the last occurrence of a specified value in a string.
    Example:-
    let text = "Hello world, welcome to the universe.";
    let result = text.lastIndexOf("world");
    console.log(result); // Output: 6

  • match()

    • The match() method is used to search a string for a match against a regular expression, and returns the matches, as an Array object.
    Example:-
    let text = "The rain in SPAIN stays mainly in the plain";
    let result = text.match(/ain/g);
    console.log(result); // Output: ain,ain,ain

  • repeat()

    • The repeat() method is used to repeat a string a specified number of times.
    Example:-
    let text = "Hello";
    let result = text.repeat(2);
    console.log(result); // Output: HelloHello

  • replace()

    • The replace() method is used to replace a specified value with another value in a string.
    Example:-
    let text = "Hello world";
    let result = text.replace("world", "Dolly");
    console.log(result); // Output: Hello Dolly

  • search()

    • The search() method is used to search a string for a match against a regular expression, and returns the position of the match.
    Example:-
    let text = "Hello world";
    let result = text.search("world");
    console.log(result); // Output: 6

  • slice()

    • The slice() method is used to extract a part of a string and return it as a new string.
    Example:-
    let text = "Hello world";
    let result = text.slice(6, 11);
    console.log(result); // Output: world

  • split()

    • The split() method is used to split a string into an array of substrings.
    Example:-
    let text = "Hello world";
    let result = text.split(" ");
    console.log(result); // Output: Hello,world

  • startsWith()

    • The startsWith() method is used to check if a string starts with a specified value.
    Example:-
    let text = "Hello world";
    let result = text.startsWith("Hello");
    console.log(result); // Output: true

  • substring()

    • The substring() method is used to extract a part of a string and return it as a new string.
    Example:-
    let text = "Hello world";
    let result = text.substring(6, 11);
    console.log(result); // Output: world

  • substr()

    • The substr() method is used to extract a part of a string and return it as a new string.
    Example:-
    let text = "Hello world";
    let result = text.substr(6, 11);
    console.log(result); // Output: world

  • toLowerCase()

    • The toLowerCase() method is used to convert a string to lowercase letters.
    Example:-
    let text = "Hello World";
    let result = text.toLowerCase();
    console.log(result); // Output: hello world

  • toUpperCase()

    • The toUpperCase() method is used to convert a string to uppercase letters.
    Example:-
    let text = "Hello World";
    let result = text.toUpperCase();
    console.log(result); // Output: HELLO WORLD

  • trim()

    • The trim() method is used to remove whitespace from both ends of a string.
    Example:-
    let text = " Hello World ";
    let result = text.trim();
    console.log(result); // Output: Hello World

  • trimStart()

    • The trimStart() method is used to remove whitespace from the beginning of a string.
    Example:-
    let text = " Hello World ";
    let result = text.trimStart();
    console.log(result); // Output: Hello World

  • trimEnd()

    • The trimEnd() method is used to remove whitespace from the end of a string.
    Example:-
    let text = " Hello World ";
    let result = text.trimEnd();
    console.log(result); // Output: Hello World
